Address Label Printing Information

Due to constraints, the following need to be observed and are important:

  • Address labels are only printed based on the list shown in the previous popup. If an address is missing, check to make sure that it's included in the DISPLAYED list below. Check your filters, search results, and the display total amount.
  • If you do NOT print the labels on the next page for any reason, the system will NOT be able to tell. As such, a note will always be logged stating "Address label scheduled to print." on all records that are printed.
  • You MUST start with a blank and full Avery 5260 address label sheet.
